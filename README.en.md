# Feng Ge Codex Pet

A high-detail chibi custom pet for the Codex desktop app and compatible Codex CLI terminals.

## Install

Download the latest `feng-ge-pet-v1.0.0.zip` from **Releases**, extract it, then run:

Windows PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

macOS or Linux:

```bash
chmod +x install.sh
./install.sh
```

Alternatively, copy the complete `feng-ge` directory to `${CODEX_HOME:-$HOME/.codex}/pets/feng-ge`. Restart Codex or choose **Settings → Pets → Refresh**, then select **峰哥**.

This is a v2 `1536 × 2288` pet atlas. The current web uploader accepts `1536 × 1872`, so the complete v2 atlas is intended for local desktop/CLI installation.

See the [official Pets documentation](https://learn.chatgpt.com/docs/pets).
